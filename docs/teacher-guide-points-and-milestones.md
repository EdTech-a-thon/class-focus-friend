# Teacher Guide: Points, Sessions, and Milestones

Class Focus Friend turns focused classroom time into a shared class goal. The class completes focus sessions, earns points, unlocks rooms, and uses points to decorate the Focus Friend's house or dress up the Focus Friend.

This guide explains how to use the system and what the class can work toward.

## Set Up a Focus Session

1. Open **Session settings** using the gear button on the Focus Session card.
2. Choose a session length: **5, 10, 15, or 20 minutes**.
3. Choose the kind of activity:
   - **Independent work:** very quiet
   - **Partner work:** moderate voices
   - **Presentation:** one clear speaker
4. If you want to use the noise meter, turn on the microphone and allow microphone access when asked. The timer can still be used without the microphone.
5. Select **Start session** when the class is ready.

The timer can be paused and continued. If the noise meter reaches the "Too loud" level, the timer pauses so the class can reset. Select **Start session** again when the class is ready to continue.

Use **Reset** if you want to return the timer to the beginning. A reset or unfinished session does not earn points and does not count as a completed session.

## How the Class Earns Points

The class earns **one point for every minute in a completed session**.

| Session length | Points earned |
| --- | ---: |
| 5 minutes | 5 points |
| 10 minutes | 10 points |
| 15 minutes | 15 points |
| 20 minutes | 20 points |

Points are added only when the timer reaches zero. Pausing does not reduce the reward, as long as the class eventually finishes the session.

The Class Progress card shows two point totals:

- **Total points earned** is the full number of points the class has earned over time.
- **Available now** is the number of unspent points the class can currently use.

Buying a reward lowers the available points, but it does not lower the total points earned.

## How Sessions Are Counted

One session is counted each time a focus timer reaches zero. Every completed session counts as **one session**, whether it lasted 5, 10, 15, or 20 minutes.

For example, completing one 5-minute session and one 20-minute session gives the class:

- **2 completed sessions**
- **25 focused minutes**
- **25 points earned**

Only completed sessions count toward opening new rooms. Paused, reset, or unfinished sessions do not count.

The app shows the most recent five sessions on the Class Progress card. It remembers up to the 20 most recent completed sessions on the same device and browser.

## What Points Can Buy

The class can spend available points in two ways. Select **Decorate room** to buy house pieces, or select **Dress up friend** to unlock accessories.

### Focus Friend Accessories

| Accessory | Cost |
| --- | ---: |
| Sunny Specs | 25 points |
| Party Hat | 50 points |
| Bow Tie | 75 points |

An accessory only needs to be purchased once. After that, it can be put on or removed at any time without spending more points.

### House Decorations

House decorations become more expensive as the class moves through the house, from **15 points** for the first decoration to **325 points** for the final one. A purchased decoration is placed in its room and cannot be purchased again.

Some decorations are part of milestones. Buying all the listed pieces for the current milestone completes it and moves the class to the next milestone.

## Buying Rooms

Rooms are purchased with points, just like decorations. The class can buy rooms in any order, but each room must be owned before it can be decorated.

| Room | Cost |
| --- | --- |
| Living Room | 100 points |
| Bedroom | 250 points |
| Kitchen | 400 points |
| Bathroom | 550 points |

The class can use **Choose room** to buy a room once it has enough points, or visit a room it already owns.

## Milestones in Order

The Class Progress card follows the milestones in the order below and shows the first milestone that still has an uncollected piece. To finish a milestone, buy every listed decoration. The class may buy a piece from a later milestone early; that piece will already count when the class reaches that milestone.

There is no extra point prize for completing a milestone. The reward is seeing the completed themed space in the Focus Friend's house and moving on to the next class goal.

Optional decorations, rooms, and Focus Friend accessories use the same available points as milestone pieces. Buying every room, decoration, and accessory costs exactly **8,000 points**: a full 160-day school year at an average of 50 points per day.

## Helpful Classroom Routine

1. Tell students the session length and the milestone piece they are working toward.
2. Choose the activity type so the noise goal matches the task.
3. Start the timer and use the noise meter as a neutral visual reminder.
4. Celebrate when the timer finishes and review the points earned.
5. Let the class save points or choose a reward together.

The app remembers progress on the same device and browser. For consistent class progress, use the same classroom computer and browser each time.
